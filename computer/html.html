HTML Concepts

1. What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It structures content on the web using elements such as headings, paragraphs, links, and images.

2. Basic Structure of an HTML Page

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>My Web Page</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
        <p>Welcome to HTML.</p>
    </body>
</html>

3. HTML Tags

HTML consists of tags that define the structure of the content. Here are some common tags:

  • <h1> - <h6> for headings
  • <p> for paragraphs
  • <a> for links
  • <img> for images
  • <ul> and <ol> for unordered and ordered lists

4. Adding Links in HTML

<a href="https://www.example.com">Click here</a>

This will create a link to "https://www.example.com".

5. HTML Forms

<form action="submit_form.php">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Submit">
</form>

This form allows the user to input their name and submit it.

6. HTML Images

<img src="image.jpg" alt="Description of image">

This will display an image named "image.jpg". The alt attribute provides a description of the image for accessibility purposes.

7. HTML Tables

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
</table>

This will create a table with two columns and a header row.

8. HTML Lists

HTML allows you to create both ordered and unordered lists:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
<ol>
    <li>First Item</li>
    <li>Second Item</li>
</ol>

9. HTML Comments

<!-- This is a comment -->

HTML comments are not visible on the webpage but can be used for documentation purposes.

10. HTML Attributes

HTML elements can have attributes, which provide additional information. For example, the href attribute specifies the URL in a link, and the src attribute specifies the source of an image.

<a href="https://www.example.com">Visit Example</a>

11. HTML Semantic Tags

HTML5 introduces semantic tags that provide meaning to the content:

  • <header> for page headers
  • <footer> for page footers
  • <article> for articles
  • <section> for sections of content

12. HTML Meta Tags

<meta charset="UTF-8">

Meta tags provide metadata about the HTML document, such as character encoding and author information.
