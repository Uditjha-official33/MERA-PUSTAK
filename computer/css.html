Complete CSS Concepts

1. What is CSS?

CSS (Cascading Style Sheets) is used to control the layout and appearance of web pages. It helps in styling HTML elements such as fonts, colors, spacing, and positioning.

2. Basic CSS Syntax

CSS consists of selectors and declarations. A declaration has a property and a value:

selector {
    property: value;
}

3. Inline CSS

Inline CSS is used to style a single HTML element directly:

<div style="color: blue;">This is a blue text.</div>

4. Internal CSS

Internal CSS is defined inside the <style> tag in the <head> section of an HTML document:

<style>
    body {
        background-color: lightgray;
    }
</style>

5. External CSS

External CSS is written in a separate file and linked to the HTML document using the <link> tag:

<link rel="stylesheet" href="styles.css">

6. CSS Selectors

CSS selectors are used to target HTML elements to apply styles. Here are some common selectors:

  • * - universal selector (applies to all elements)
  • element - targets all instances of an element (e.g., p, h1)
  • #id - targets an element by its id
  • .class - targets elements by their class

7. CSS Colors

CSS allows you to define colors using names, hex values, RGB, and HSL:

p {
    color: red;  /* Named color */
}

h1 {
    color: #FF5733;  /* Hexadecimal color */
}

h2 {
    color: rgb(255, 87, 51);  /* RGB color */
}

8. CSS Box Model

The CSS box model describes the structure of elements. It consists of margins, borders, padding, and content:

div {
    margin: 20px;
    padding: 10px;
    border: 1px solid black;
}

9. CSS Positioning

CSS provides several ways to position elements on a page:

div {
    position: relative;
    top: 20px;
    left: 50px;
}
  • relative - positions an element relative to its normal position
  • absolute - positions an element relative to its nearest positioned ancestor
  • fixed - positions an element relative to the viewport
  • sticky - makes an element "stick" when you scroll past it

10. CSS Flexbox

Flexbox is a layout model that allows items within a container to be aligned easily:

container {
    display: flex;
    justify-content: center;
    align-items: center;
}

11. CSS Grid

CSS Grid is a powerful layout system for creating two-dimensional layouts:

container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

12. CSS Pseudo-Classes

Pseudo-classes target elements in specific states. Examples include:

a:hover {
    color: red;
}

input:focus {
    border-color: blue;
}

13. CSS Pseudo-Elements

Pseudo-elements allow you to style specific parts of an element:

p::first-letter {
    font-size: 2em;
    color: blue;
}

14. CSS Transitions

CSS transitions allow you to change property values smoothly over a given duration:

button {
    transition: background-color 0.3s ease;
}

button:hover {
    background-color: blue;
}

15. CSS Animations

CSS animations enable keyframe-based animations:

@keyframes example {
    0% {
        transform: scale(1);
    }
    100% {
        transform: scale(1.5);
    }
}

div {
    animation: example 2s infinite;
}

16. Media Queries

Media queries allow you to apply styles based on the viewport size:

@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}
