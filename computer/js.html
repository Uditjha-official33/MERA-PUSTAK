Complete JavaScript Concepts

1. What is JavaScript?

JavaScript is a high-level, interpreted programming language used to create interactive effects within web browsers. It allows dynamic updates, controls multimedia, and supports various APIs.

2. Basic JavaScript Syntax

JavaScript code consists of statements, variables, operators, and functions. Here's an example of basic syntax:

let message = "Hello, World!";
console.log(message);  // Output: Hello, World!

3. Variables in JavaScript

You can declare variables using let, const, or var:

let name = "John";
const age = 30;
var city = "New York";

Additional Example: You cannot reassign a value to a const variable, but you can reassign a let or var variable.

let count = 10;
count = 20;  // Reassigned

const pi = 3.14;
pi = 3.14159;  // Error: Assignment to constant variable

4. Data Types in JavaScript

JavaScript supports various data types such as strings, numbers, booleans, arrays, and objects:

let name = "Alice";  // string
let age = 25;         // number
let isStudent = true; // boolean
let colors = ["red", "blue", "green"]; // array
let person = { name: "Bob", age: 30 };  // object

Additional Example: JavaScript also has undefined and null types.

let unassigned;
console.log(unassigned); // Output: undefined

let empty = null;
console.log(empty); // Output: null

5. Functions in JavaScript

Functions allow you to group reusable code. They are defined using the function keyword:

function greet(name) {
    console.log("Hello, " + name);
}

greet("Alice"); // Output: Hello, Alice

Additional Example: Functions can return values:

function add(a, b) {
    return a + b;
}

let result = add(2, 3);
console.log(result); // Output: 5

6. Loops in JavaScript

Loops allow repetitive tasks to be automated. The most common types are for, while, and do...while:

for (let i = 0; i < 5; i++) {
    console.log(i); // Output: 0, 1, 2, 3, 4
}

Additional Example: while loop:

let i = 0;
while (i < 5) {
    console.log(i); // Output: 0, 1, 2, 3, 4
    i++;
}

7. Conditionals in JavaScript

Conditionals like if, else if, and else help you make decisions based on conditions:

let age = 20;

if (age >= 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}

Additional Example: else if:

let time = 14;
if (time < 12) {
    console.log("Good Morning");
} else if (time < 18) {
    console.log("Good Afternoon");
} else {
    console.log("Good Evening");
}

8. Objects in JavaScript

Objects allow you to group related data together. They are defined using curly braces {}:

let person = {
    name: "John",
    age: 30,
    greet: function() {
        console.log("Hello, " + this.name);
    }
};

person.greet(); // Output: Hello, John

Additional Example: Accessing object properties:

let person = {
    name: "Alice",
    age: 25
};

console.log(person.name);  // Output: Alice
console.log(person["age"]); // Output: 25

9. Arrays in JavaScript

Arrays are used to store multiple values in a single variable:

let fruits = ["apple", "banana", "cherry"];
console.log(fruits[1]); // Output: banana

Additional Example: Modifying arrays:

fruits.push("orange");
console.log(fruits); // Output: ["apple", "banana", "cherry", "orange"]

fruits.pop();
console.log(fruits); // Output: ["apple", "banana", "cherry"]

10. Events in JavaScript

Events allow you to interact with the webpage, like clicking a button. Event handlers are used to manage them:

document.getElementById("myButton").addEventListener("click", function() {
    alert("Button clicked!");
});

11. DOM Manipulation in JavaScript

The Document Object Model (DOM) allows you to access and modify HTML elements:

document.getElementById("myDiv").innerHTML = "Hello, World!";

12. JavaScript ES6 Features

ES6 introduced many new features, including let, const, arrow functions, and template literals:

const greet = (name) => {
    return `Hello, ${name}`;
};

console.log(greet("Alice")); // Output: Hello, Alice

13. Promises in JavaScript

Promises handle asynchronous operations in JavaScript:

let promise = new Promise(function(resolve, reject) {
    let success = true;
    if (success) {
        resolve("Operation was successful");
    } else {
        reject("Operation failed");
    }
});

promise.then(function(result) {
    console.log(result); // Output: Operation was successful
}).catch(function(error) {
    console.log(error);
});

14. Async/Await in JavaScript

Async/await simplifies working with promises:

async function fetchData() {
    let response = await fetch('https://api.example.com');
    let data = await response.json();
    console.log(data);
}

fetchData();

15. JavaScript Error Handling

JavaScript provides error handling using try, catch, and finally:

try {
    let result = riskyFunction();
} catch (error) {
    console.log("Error occurred:", error);
} finally {
    console.log("This always runs");
}

16. JavaScript Modules

Modules allow you to split code into smaller files:

export function greet(name) {
    console.log("Hello, " + name);
}

import { greet } from './greet.js';
greet("Alice");
